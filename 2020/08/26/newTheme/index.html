回忆过去

11 年 7 月份，我第一次接触博客这个概念。
淘米校巴，这个名词算是我童年的一部分。
只不过校巴火得快，倒得也快，我小学毕业的那个暑假，校巴逐渐淡出了我的视野。

人是孤独的。
人是需要认同的。

直至今日，我玩游戏仍然喜欢 MMORPG。这类游戏带来的存在感、归属感、真实感，不是其他种类游戏，甚至不是其他种类娱乐方式可以给予的。
网络世界、游戏世界中，我可以既是个语言的巨人，也是个行动的巨人。但是现实中，充其量是个幻想的巨人。

人是会变的。
小时候害怕孤独，长大了反而觉得一个人很有意思。

我个人以为，凡是不再寻找存在感并且开始注重自己的，都会对生活记录感兴趣。
没有好朋友，若是记性再差一点，估计自己活着的时候干过什么事，之后都记不住了。
记录下来是一个好方法。

一年前到半年前

19 年 4 月份，我第一次看到感动我自己的博客：Angel_Kitty
怎么说呢，我是没想过 cnblog 里面会有讲知识如此细致的博主。
（如果说这就是感动我的标准的话，也可以说我的标准很低）

那天我坐在图书馆 9 楼的空调前，正在准备建模校赛… 那个场景现在也忘不了。

再到今年 1 月份我又机缘巧合逛到了博主写的另一篇技术博客。
由于她的 UI 实在是太特殊了，在 cnblog 里面算是一股清流了…
（这么看，我对 清流 的定义有很怪）

看到博主她非计算机专业跨考清华以及对自己大学四年的记录时，我为这种坚韧的精神所折服。
再快进到 bydl 5 月份的题解，我当时觉得，我也有东西可记录。
也就是这个时候，才开始考虑记录一点点关于我的事情。

landscape -> NexT -> butterfly

bydl 现在的 theme 是 NexT，但是当时是 landscape，我现在想来，还是这个默认主题好看，NexT 属实有点拉胯。后来嘛，就换成了 GitHub 上 star 最多的 NexT。
但是越看越不顺眼哈，在昨晚在 GitHub 上翻了一凌晨的 hexo-theme 之后（奇怪的定语），最终选择了 butterfly。

不得不说，我是觉得每一个 theme 符合我的心意的，butterfly 只能算是矬子里拔大个。但是我也不会啊，这波是我上我不行。

$ skyleaworlder.github.io restart

很好，之前的浏览量和评论都清空了。
重新开始吧。
